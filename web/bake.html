薛之谦（Joker Xue），1983年7月17日出生于上海，华语流行乐男歌手、影视演员、音乐制作人，毕业于格里昂酒店管理学院。
2005年，因参加选秀节目《我型我秀》而正式出道[1]  。2006年，发行首张同名专辑《薛之谦》，随后凭借歌曲《认真的雪》获得广泛关注[2-3]  。2007年，凭借专辑《你过得好吗》获得MusicRadio中国TOP排行榜内地年度最受欢迎男歌手奖。2008年，发行专辑《深深爱过你》，同年在上海举行个人首场演唱会“谦年传说”[4-5]  。2013年，专辑《几个薛之谦》获得MusicRadio中国TOP排行榜内地推荐唱片[6]  。2014年，凭借专辑《意外》获得第21届东方风云榜颁奖最佳唱作人[7]  ；同年他还获得音悦V榜年度盛典年度创作歌手奖[8]  。
2015年6月，薛之谦首度担当制作人并发行原创EP《绅士》[9]  ；同年，他还主演都市励志剧《妈妈像花儿一样》[10]  。2016年，凭借EP《绅士》、《一半》获得第16届全球华语歌曲排行榜最受欢迎男歌手、五大最受欢迎男歌手奖、上海地区杰出歌手奖及最受欢迎创作歌手奖，而歌曲《初学者》则获得年度二十大金曲奖[11-12]  。2017年4月，开启“我好像在哪见过你”全国巡回演唱会[13]  ；同年，薛之谦获得第7届全球流行音乐年度盛典年度最佳男歌手及MusicRadio榜中国TOP排行榜内地最佳男歌手[14-15]  。

薛之谦早年经历

编辑
薛之谦4岁的时候，薛妈妈因为风湿性心脏病去世，薛之谦由爸爸带大。高中毕业的时候，爸爸为了筹钱支持薛之谦去瑞士留学把房子卖了，用卖房所得的60万送他到瑞士留学，学习酒店管理。放假回家的薛之谦在商场门口被经纪公司的星探看中，他们百般动员他留下当艺人，答应给他签约、出唱片。可等到薛之谦决定暂停瑞士的学业留下以后，经纪公司又要他出30万的包装费用。薛之谦灰心之余，又遇上了真人版《我为歌狂》剧组在招聘演员，他很幸运地获得了其中一个角色，可遗憾的是最终由于版权问题，这部戏没有成功开拍[16-17]  。

薛之谦演艺经历

编辑
平面照(18张)
2005年，薛之谦因参加选秀节目《我型我秀》获得四强而正式出道；同年9月，签约上腾娱乐[18]  。
2006年6月12日，发行首张个人原创同名专辑《薛之谦》，他包揽专辑六首曲和五首词的创作，专辑发行1个月销量突破20万张，而他则成为年度销量最高内地男歌手，随后，主打歌《认真的雪》获得第十四届东方风云榜十大金曲奖[2]  [20]  。
2007年7月31日，发行第二张个人原创专辑《你过得好吗》，该专辑销量在一个月内破15万，而薛之谦凭此专辑荣获MusicRadio中国TOP排行榜内地年度最受欢迎男歌手奖，同名主打歌《你过得好吗》则获得北京流行音乐典礼内地年度金曲奖[21-22]  。
2008年11月26日，发行第三张个人原创专辑《深深爱过你》，专辑收录9首歌曲，同名主打歌《深深爱过你》更是由他作词作曲并演唱，分为前世、今生两个版本，随后，该曲获得北京流行音乐典礼年度金曲奖及第9届全球华语歌曲排行榜最受欢迎二十大金曲奖[4]  [24]  ；12月20日，薛之谦在上海举行个人首场演唱会“谦年传说”[5]  。
2009年9月21日，推出单曲《未完成的歌》[25]  ；11月30日，薛之谦获得中国原创歌曲奖内地最佳舞台演绎奖和内地最佳专辑奖[26]  ；同年，发行精选集《未完成的歌》。
2010年9月，薛之谦跨界影视，主演了个人首部大银幕作品《一只狗的大学时光》[27]  。
专辑《意外》(5张)
2011年3月14日，推出单曲《我终于成了别人的女人》[28]  ；随后，该曲获得第11届全球华语歌曲排行榜金曲奖，而薛之谦亦则获得上海地区杰出歌手奖，该曲还于次年4月7日获得第十九届东方风云榜十大金曲[29-31]  。
2012年7月17日，发行的第四张个人专辑《几个薛之谦》共收录9首歌曲，专辑于次年4月获得MusicRadio中国TOP排行榜内地推荐唱片[6]  ；9月，他与上腾娱乐约满不续[33]  ；12月27日，推出单曲《方圆几里》[34]  。
2013年1月6日，正式签约坤宏传媒[33]  ；10月5日，薛之谦获得全球华语歌曲排行榜上海地区杰出歌手[35]  ；11月11日，发行第五张个人专辑《意外》，专辑收录10首歌曲[36]  。
2014年3月31日，凭借专辑《意外》获得第21届东方风云榜颁奖最佳唱作人，而专辑同名歌曲亦获得十大金曲[7]  ；4月15日，他还获得音悦V榜年度盛典年度创作歌手奖[8]  。
绅士(7张)
2015年6月8日，发行的原创EP《绅士》共收录3首歌曲，这是他出道十年首次担当制作人，包揽了该EP的所有词曲[9]  ；6月11日，正式签约海蝶音乐[37]  ；10月26日，担任制作人推出第二张个人原创EP《一半》[38]  ；随后，搭档朴海镇、张亮主演都市爱情剧《男人帮·朋友》，他在剧中饰演贸易公司小老板高益[39]  ；10月5日，其主演的都市励志剧《妈妈像花儿一样》播出，薛之谦在剧中饰演花花少爷张来福，并为该剧配唱片尾曲及多首插曲[10]  。
2016年3月29日，薛之谦凭借EP《绅士》、《一半》获得第二届酷音乐亚洲盛典年度最受欢迎唱作人及年度电视音乐金曲艺人[40]  ；4月15日，获得第二十届全球华语榜中榜最受欢迎唱作歌手[41]  ；5月17日，发行EP《初学者》，同时同名歌曲《初学者》成为爱情喜剧片《A测试之爱情大冒险》的主题曲[42-43]  ；6月4日，作为固定嘉宾加盟职业体验真人秀《挑战者联盟第二季》[44]  ；6月7日，发行原创单曲《刚刚好》[45]  ；随后，还推出个人原创单曲《我好像在哪见过你》，该曲成为魔幻动画片《精灵王座》的主题曲；30日，薛之谦作为中方艺人加入亚洲首档中韩人气偶像歌艺秀《作战吧偶像》[46]  ；7月2日，加盟儿童歌唱类节目《中国新声代第四季》[47]  ；7月17日，他在北京举办“初学者”免费演唱会；
薛之谦2017年新歌宣传图(3张)
7月18日，发行专辑《初学者》[42]  ；10月19日，薛之谦获得第十六届全球华语歌曲排行榜最受欢迎男歌手、五大最受欢迎男歌手奖、上海地区杰出歌手奖及最受欢迎创作歌手奖，而歌曲《初学者》则获得年度二十大金曲奖[11-12]  ；11月9日，音乐爱情题材电影《音乐江湖》上映，他在片中饰演性格内向、对音乐执着的学生陈浩，并配唱主题曲《我们爱过就好》[48]  ；28日，推出单曲《火星人来过》[49]  ；12月10日，歌曲《初学者》再次获得音乐先锋榜年度最佳原创歌曲奖[50]  ；18日，加盟户外明星挑战类真人秀节目《我们的挑战》[51]  。
2017年3月17日，发行单曲《高尚》[52]  ；4月1日，其零片酬出演的喜剧电影《有完没完》上映，他在片中饰演痴情轻生男和平[53]  ；随后，他还与杨幂、华晨宇一起担任首席星推官，参与偶像养成类娱乐选秀节目《明日之子》[54-55]  ；4月8日，加盟音乐竞技综艺节目《金曲捞》[56]  ；4月22日，“我好像在哪见过你”全国巡回演唱会首站在大连开启[13]  ；5月7日，薛之谦获得第7届全球流行音乐年度盛典年度最佳男歌手，专辑《初学者》亦获得年度最受欢迎专辑奖，而专辑同名歌曲《初学者》则获得年度二十大金曲[14]  ；6月20日，以歌手的蜡像造型成为上海杜莎夫人蜡像馆首位进驻杜莎音乐梦工厂的艺人[58]  ；随后，他还获得MusicRadio榜中国TOP排行榜内地最佳男歌手[15]  ；7月9日，与TFBOYS共同主演的青春励志成长校园剧《我们的少年时代》首播，薛之谦在剧中饰演废柴教师陶西[59]  ；7月19日，获得亚洲金曲大赏年度内地最佳男歌手和现场人气王两个奖项[60]  ；8月27日，薛之谦获得亚洲新歌榜最佳男歌手，同时《初学者》也获得最佳专辑奖[61]  。

薛之谦个人生活

编辑
2012年，薛之谦的火锅店“上上谦”开业，截止至2016年，该火锅店已扩张到五家店[62]  。在经营火锅店的同时，他从火锅店挤出50万元创立独立品牌在淘宝开中端女装店，因在网上销售良好，女装店转战线下开实体店[63]  。
2012年，薛之谦与高磊鑫结婚，两人于2015年9月离婚[64]  。
2017年9月8日，薛之谦和高磊鑫通过微博宣布复合[65]  。

薛之谦社会活动

编辑
2007年初，薛之谦作为流行音乐的代表，和其他艺术家一起前往俄罗斯，参加由中华人民共和国文化部和俄罗斯联邦文化电影署联合举办的中国文化节重点项目”沿伏尔加河友谊之旅”，在沿河11个城市作访问演出[80]  ；8月，专辑《你过得好吗》加入公益活动中，部分专辑在网上进行拍卖，所得金额全数捐赠给失聪的小朋友[81]  。
2008年1月20日，薛之谦献声赛我网“幸福之翼”公益晚会[82]  ；随后，他当选火炬手，在河南省郑州市传递火炬[83]  ；3月，在上海举行新书发布会，他除了自己捐出部分稿费外，还拿出10张照片进行拍卖，所得款项通过上海市红十字会捐给雪灾灾区，支援灾区重建[84]  。
2010年4月，他无酬劳出席公益晚会《抗旱救灾我们在行动》，同时自行负担行程和费用[85]  。
2011年9月16日，献唱蒙牛“生态行动 助力中国”生态公益行动的收官演唱会[86]  。
2012年4月，为电影《星星的孩子》做首映票义卖活动[87]  ；7月，他陪一位老奶奶等高价回收废品的人，一个小时那个人还是没来，薛之谦就帮老奶奶背着三个大塑料袋，帮老奶奶背回了家，并发微博呼吁大家一起帮助老人[88]  。
2013年1月，现身风尚志举办的慈善晚宴现场，以一万两千元的价格拍得吴秀波亲自手绘的化妆包，并将所得捐赠给嫣然天使儿童基金；随后，薛之谦在参加“青年志愿者公益之夜”活动时，将伴随自己多年的吉他赠送给一位爱音乐的盲童[89]  。
2016年3月，参加启动“美肤宝阳光计划”，提倡暖心支持公益，关爱留守儿童[90]  ；2016年7月，作为“援甘”成员加入韩红发起“韩红爱心，百人援助”的大型慈善公益活动，他不仅默默向此次慈善公益活动捐赠资金，还在韩红的邀请下现身最前线[91]  ；8月13日，被聘请为“爱飞翔·乡村教育项目”的首席梦想官，并参加第十届“爱飞翔·乡村教育项目”之乡村教师培训公益活动[92]  。
2017年6月，薛之谦与周迅参加“ONE NIGHT给小孩”公益探访来到上海某学校，在世界羽毛球冠军、国际特殊奥林匹克东亚区羽毛球讲师卢兰的介绍下，深入了解智力障碍儿童群体的日常学习和生活并进行亲切互动，他还希望以后能用行动的力量更深入的帮助这些孩子们[93-94]  ；7月20日，薛之谦以“世界和平”造型入驻重庆杜莎夫人蜡像馆，揭幕活动现场他携手重庆杜莎为6名山城贫困学子送上“高考圆梦”资助公益行动，帮助这些孩子顺利完成大学学业[95]  。

薛之谦人物评价

编辑
从《认真的雪》、《深深爱过你》再到《几个你》、《我知道你都知道》和
灵魂歌者(27张)
《方圆几里》，深情款款的“薛氏情歌”是他的不变招牌，其中《黄色枫叶》、《认真的雪》等招牌创作，既迎合了市场的喜好，又是他自己擅长的情歌类型，这些歌帮助他积累了人气，创作才子的名号也由此沿袭下来（贵阳黔讯网评[108]  ）。
大男孩的顽皮、成熟男人的优雅绅士，光影中的薛之谦亦是如此，歌手、演员、主持人，拥有多重身份的他，就像他的文字一样，简单而又纯粹（精品购物指南评[109]  ）。
手握十几档节目，除了长微博外，随随便便一篇普通微博点赞量也几十万，但这也证明在更多人眼里他还是一个有智慧的谐星以及收视担当。只要看过他上的节目会发现，一旦有让他唱歌的机会，无论是什么样的，他都不会放过。见过薛之谦的人都知道，他有种自然生成的幽默感，这也许注定了他会成为杰出的段子手（腾讯网评[110]  ）。
